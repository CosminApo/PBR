READ ME FILE

A Physically Based Renderer based on the Cook-Torrance BRDF equation developed using OOP C++ and OpenGL.

//////////////////////////////////////////////////////////////////////////////////////////
PBR PROGRAM

To run the program, go to Code/Release/ and run GLSandbox.exe. To close the program, click on X button on the console.

Inside the program you have control of the camera and of the model.

CAMERA CONTROLLS:
W -> move camera forward
A -> move camera left
S -> move camera backwards
D -> move camera right

OBJECT CONTROLLS:
Up arrow    -> move the object up
Left arrow  -> move the object left
Down arrow  -> move the object down
Right arrow -> move the object right
